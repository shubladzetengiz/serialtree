# SerialManager

[🇬🇪 ქართული](README.md) · [🇬🇧 English](README.en.md) · [🇷🇺 Русский](README.ru.md) · [🇫🇷 Français](README.fr.md)

Веб-приложение для управления сериалами и фильмами. PHP + SQLite + Tailwind CSS.

![Демонстрация](demo.png)

## Требования

- PHP 8.1 или новее
- Расширение `mbstring` (часть PHP)
- Пакет `php-sqlite3`

```
sudo apt install php-sqlite3 php-mbstring
```

## Запуск

### С роутером (рекомендуется)

```sh
php -S localhost:8002 -t serial2 serial2/router.php
```

### Без роутера

```sh
php -S localhost:8002 -t serial2
```

Роутер обрабатывает раздачу статических файлов (JS, CSS, изображения).

### Смена порта

Замените `8002` на любой свободный порт.

## Возможности

### Основные
- Добавление / Редактирование / Удаление сериалов и фильмов
- Два режима отображения: сетка и список
- Живой поиск (Unicode, регистронезависимый, 300ms debounce)
- Фильтр по статусу
- Сортировка: по дате (новые/старые) и по алфавиту (А–Я / Я–А)

### Обложки
- Скачивание по внешнему URL и сохранение локально
- Бейдж: ✓ Local / ✗ Online
- Более 100 обложек уже загружено

### Авто-статус
- Оба поля заполнены (сезон + эпизод) → `ნანახი` (Просмотрено)
- Заполнено только одно поле → `გასაგრძელებელია` (В процессе)
- Оба пусты → `სანახავია` (К просмотру)
- Сезон > 99 → очищает оба поля, статус → `სანახავია`

### Импорт / Экспорт
- SQL экспорт (один файл для обеих таблиц)
- SQL импорт (.sql, макс 10MB, CSRF-защита)
- Импорт фильтрует колонки под существующую схему (обратная совместимость)

### Интерфейс
- 4 языка: 🇬🇪 Грузинский, 🇬🇧 Английский, 🇷🇺 Русский, 🇫🇷 Французский
- Тёмная тема (Tailwind CSS + Font Awesome 6)
- Адаптив: таблица на десктопе, карточки на мобильных
- CSRF-защита (POST-запросы)
- Удаление с 5-секундным таймером

## База данных

- Файл: `data.sqlite`
- Таблицы: `series` (сериалы) и `movies` (фильмы)
- Автомиграция: отсутствующие колонки добавляются автоматически

### series
| Колонка | Тип | Описание |
|---|---|---|
| id | INTEGER | Первичный ключ |
| cover | TEXT | Путь/URL обложки |
| title | TEXT | Название |
| season | INTEGER | Сезон |
| episode | INTEGER | Эпизод |
| status | TEXT | Статус |
| rating | INTEGER | Оценка (0–5) |
| resource_url | TEXT | Ссылка на источник |

### movies
| Колонка | Тип | Описание |
|---|---|---|
| id | INTEGER | Первичный ключ |
| cover | TEXT | Путь/URL обложки |
| title | TEXT | Название |
| description | TEXT | Описание |
| status | TEXT | Статус |
| rating | INTEGER | Оценка (0–5) |
| resource_url | TEXT | Ссылка на источник |

## Восстановление

Если `data.sqlite` отсутствует (или при первом запуске), приложение предлагает:
1. **Создать пустую БД** — автоматическая миграция
2. **Импортировать SQL файл** — восстановление из бэкапа
3. **Отмена**

Чтобы создать SQL бэкап: нажмите кнопку `Экспорт` на панели инструментов.

## Безопасность

- CSRF-токен на всех POST-запросах
- Защита от SQL-инъекций: prepared statements
- Защита от XSS: `htmlspecialchars()` на всех выводах
- Импорт файлов: только `.sql`, макс 10MB
